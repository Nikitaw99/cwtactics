game.setNextPlayerAsTurnOwner()

Sets the next available player as turn owner. The search algorithm starts with the id of the current turn owner +1. If the player which owns this id is not alive, another +1 will be added to the searched id. If the last id of the player list is also not alive the search will start with 0 again. The day counter increases if this situation happens.
