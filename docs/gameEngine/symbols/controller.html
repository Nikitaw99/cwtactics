Namespace Detail
controller

This is the main access layer for the custom wars tactics game client. All data changing actions will be invoked from this layer.

The layer itself is build as state machine which represents a player action. Every action starts by a selection of a tile. Which the selected object will be choosen by the state of the tile. An empty tile leads to a map action. An empty (owned) property leads to a property actions like buying an unit. The last option will be choosen if the tile is occupied by an own unit.

Action type search:

  • 1. tile occupied by an actable unit
    if yes then unit action else 2.
  • 2. is tile a property that can build things
    if yes then property action else 3.
  • 3. map action
