Defined in: engineBase.js.

Namespace Detail
util

Some useful utility functions are stored in this layer. This layer contains the logging functions of custom wars tactics. These functions are overwritable to have a custom log behaviour for the game client. As example if you use a graphical logging solution like BlackbirdJs.
