status

Status of the logger, the status describes want level of debug messages will be logged. If a message is on a lower level as the set status of the debug logger, then the message won't be displayed on the console.

Value can be one of the following constants:
meow.Logger.LEVEL_OFF
meow.Logger.LEVEL_INFO
meow.Logger.LEVEL_FINE
meow.Logger.LEVEL_WARN
meow.Logger.LEVEL_CRITICAL
