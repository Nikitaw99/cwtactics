<static>
meowEngine.Class(extendClass, classImpl)

Creates a new class with a given class implementation

// simple classes
var myClass1 = meow.Class({ x : alert("Hello1") });
var myClass2 = meow.Class({ x : alert("Hello2"), y : alert("Bye2") });
var myClass3 = meow.Class({ x : alert("Hello3") });

// extend classes

// myClass4.x => alert("Hello1")
// myClass4.Super => myClass1
var myClass4 = meow.Class(myClass1,{...});

// myClass4.x => alert("Hello1"), myClass4.y => alert("Bye2")
// myClass4.Super => myClass1
var myClass4 = meow.Class(myClass1,myClass2,{...});
Fork of http://myjs.fr/my-class/ 04.06.2011
Current State: No changes
Parameters:
{Object...} extendClass
class(es) that will be extended. Is a variable argument, so you can set as many classes you want. The first class you def., will be the super class, the other child classes of a deeper level. You only can access the first class with the Super attribute. Parsing algorithm is from the outer right subclass, going left to the first class argument. (should be meow classes)
{Object} classImpl
class implementation
