Class meowEngine.CommandStack


Defined in: CommandSystem.js.

Class Detail
meowEngine.CommandStack()

Command stack is a class for holding commands. It allows a revision based handling of the commands including moving between single revisions etc.

Method Detail
commit(cmd, execute)

Commits a command to the command stack and executes it, if wanted by the executor of this function. If the command stack is not at the head state, this function will first goto the head revision and execute all commands, before it processes the command. If you want to drop all changes between the current revision and the head revision, then use markHead() first.

Parameters:
{Command} cmd
command instance, that will be added to the stack
{Boolean} execute
if true, the command will executed, all otherwise ( like undefined ) it won't be executed
gotoHead()

Goes strictly to the active head revision. All revisions between the active revision and the head revision will be executed.

gotoRev(stateNum)

Brings the state to a given revision. If the target revision is before the active revision, all revisions between same will be undone. If the target revision is after the active revision, all revisions between them will be executed.

Parameters:
{Number} stateNum
revision number
{Number} headRev()
Returns:
{Number} head revision
markHead()

Marks the current active revision as the head revision and drops all revisions after the new head revision from the timeline.
